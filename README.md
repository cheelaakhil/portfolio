# Cheela Akhil — Portfolio

A dark, minimal portfolio built with React + TypeScript + Vite.

## Getting Started

```bash
npm install
npm run dev
```

Then open http://localhost:5173

## Build for Production

```bash
npm run build
npm run preview
```

## Customization

All your personal data is in one place: `src/config.ts`

Update the following fields:
- `links.linkedin` — your LinkedIn profile URL
- `links.github` — your GitHub profile URL
- `links.leetcode` — your LeetCode profile URL
- `bio` — your bio paragraphs
- `projects` — add/edit projects
- `experience` — add/edit experience entries
- `techRows` — update your tech stack

## Stack

- React 18 + TypeScript
- Vite 5
- Space Grotesk + Syne (Google Fonts)
- Pure CSS animations (no GSAP dependency needed)
- IntersectionObserver for scroll reveals
